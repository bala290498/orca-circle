# Images Folder

This folder contains the following image assets:

## Required Images

- **og-image.png** - Open Graph image for social media sharing (recommended size: 1200x630px)
- **logo.png** - Logo image for the Trust Reach Club brand (recommended size: varies based on usage)

## Usage

- `og-image.png` is referenced in `app/layout.tsx` for Open Graph metadata
- `logo.png` is used in `components/Header.tsx` for the site logo

Please add these image files to this folder.
